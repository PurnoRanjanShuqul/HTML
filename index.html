Document Foundation

Must Master

HTML page-এর মূল skeleton এখানে থাকে।

Highest Priority

1. Basic Document Structure

doctype, html, head, body, title, meta, link, script
Short Detail

একটি HTML file কীভাবে শুরু হয়, browser কীভাবে page বুঝে, CSS/JS কোথায় যুক্ত হয়—সব এখানে।

Example
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My Page</title>
</head>
<body>
  Hello
</body>
</html>

Text & Content

Must Master

Text structure clean না হলে কোনো page professional লাগে না।

Highest Priority

2. Text Content Tags

h1-h6, p, strong, em, br, hr, span, div
Short Detail

Heading, paragraph, importance, emphasis, inline text targeting এবং content grouping এর জন্য।

Key Focus
  • h1 সবচেয়ে important heading
  • div = block container
  • span = inline container
  • strong/em semantic meaning দেয়
Example
<h1>Main Title</h1>
<p>This is <strong>important</strong> text.</p>
<p>I <em>really</em> enjoy HTML.</p>

Navigation & Media Basics

Must Master

User navigation এবং visual content — দুটোই basic web skill।

Highest Priority

3. Links and Images

a, img, href, src, alt
Short Detail

Page navigation-এর জন্য link, visual content-এর জন্য image use হয়।

Key Focus
  • href = কোথায় যাবে
  • src = file source
  • alt = image description
Example
<a href="about.html">About</a>
<img src="photo.jpg" alt="A student learning HTML">

Lists & Tables

Must Master

Structured information properly দেখাতে lists এবং tables লাগে।

Highest Priority

4. Lists

ul, ol, li
Short Detail

Bullet বা numbered item group দেখানোর জন্য।

Example
<ol>
  <li>Learn HTML</li>
  <li>Practice Daily</li>
</ol>
Highest Priority

5. Tables

table, tr, th, td, thead, tbody, caption
Short Detail

Row-column data দেখানোর জন্য table use করা হয়।

Example
<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Rahim</td>
    <td>20</td>
  </tr>
</table>

Forms & Input System

Must Master

HTML expert হতে forms খুব ভালোভাবে জানা লাগবে।

Highest Priority

6. Core Form Elements

form, input, label, button, textarea, select, option
Short Detail

User data collect করার জন্য এই elements use হয়।

Example
<form>
  <label for="email">Email</label>
  <input id="email" type="email">
  <button type="submit">Send</button>
</form>
Highest Priority

7. Important Input Types

text, email, password, number, tel, file, radio, checkbox, date, range, color
Short Detail

Different kind of data নেওয়ার জন্য different input type ব্যবহার হয়।

Example
<input type="text">
<input type="email">
<input type="checkbox">
<input type="date">
Very Important

8. Form Submission & Structure

action, method, enctype, fieldset, legend, datalist
Short Detail

Form কোথায় যাবে, কীভাবে submit হবে, আর form group কিভাবে organize হবে তা এখানে শিখবে।

Example
<form action="/submit" method="post">
  <fieldset>
    <legend>Personal Info</legend>
    <input type="text">
  </fieldset>
</form>

Attributes

Must Master

Tag-এর power আসে attributes থেকে। এগুলো আলাদা category হিসেবে mastery করা দরকার।

Highest Priority

9. Core Attributes

href, src, alt, id, class, title, style
Short Detail

Link, image, element targeting, title, quick styling — এগুলোর জন্য সবচেয়ে common attributes।

Example
<a href="about.html" class="nav-link">About</a>
<img src="logo.png" alt="Website logo">
Highest Priority

10. Form Attributes

name, value, placeholder, type, required, readonly, disabled, checked, selected
Short Detail

Form field এর behavior control করতে use হয়।

Example
<input type="text" name="username" placeholder="Enter name" required>
<input type="checkbox" checked>
Important

11. Global Attributes

hidden, tabindex, lang, dir, data-*
Short Detail

অনেক tag-এ use করা যায় এবং accessibility/custom behavior-এ useful।

Example
<p lang="bn" data-topic="html">বাংলা লেখা</p>
Important

12. Useful Special Attributes

target, rel, download, width, height, loading
Short Detail

Image optimization, file download, new tab, external links নিরাপদভাবে handle করতে useful।

Example
<a href="file.pdf" download>Download</a>
<img src="photo.jpg" alt="Photo" loading="lazy" width="300">

Semantic HTML

Very Important

এই section-এ শুধু meaning-based layout structure রাখা হয়েছে। Forms বা text details এখানে repeat করা হয়নি।

Very Important

13. Page Layout Semantics

header, nav, main, footer
Short Detail

পুরো page-এর major layout areas define করে।

Example
<header>Header</header>
<nav>Menu</nav>
<main>Content</main>
<footer>Footer</footer>
Very Important

14. Content Semantics

section, article, aside, figure, figcaption
Short Detail

Related content, standalone content, side info, captioned media — এগুলো meaningfulভাবে show করে।

Example
<section>
  <article>Blog content</article>
  <aside>Related links</aside>
</section>

Accessibility

Very Important

এই section-এ শুধু accessibility rules রাখা হয়েছে — tags repeat না করে principles-based explanation।

Must Follow

15. Core Accessibility Rules

alt text, labels, heading order
Short Detail

Image-এ alt, form-এ label, এবং logical heading order maintain করা accessibility-এর foundation।

Example
<label for="name">Name</label>
<input id="name" type="text">
<img src="user.jpg" alt="User profile photo">
Important

16. Use the Right Element for the Right Job

button for actions, anchor for navigation
Short Detail

যা click করলে অন্য page-এ যায় তা link, আর যা action trigger করে তা button হওয়া উচিত।

Example
<a href="about.html">Go to About</a>
<button type="button">Open Menu</button>

SEO Basics

Important

SEO section-এ শুধু search visibility related HTML রাখা হয়েছে।

High Impact

17. Title and Meta Description

title, meta description
Short Detail

Page topic search engine-কে বোঝাতে এবং search result snippet improve করতে সাহায্য করে।

Example
<title>Learn HTML in Bangla</title>
<meta name="description" content="Complete HTML learning guide in Bangla">
Important

18. SEO-Friendly Structure

clear h1, logical headings, semantic layout, language declaration
Short Detail

Clear structure থাকলে search engine content context ভালোভাবে বুঝতে পারে।

Example
<html lang="en">
<h1>HTML Form Tutorial</h1>
<h2>Input Types</h2>

Advanced Useful Tags

Better to Know

এই section-এ advanced tags রাখা হয়েছে যেগুলো essential না হলেও expert polish দেয়।

Useful Advanced

19. Interactive Utility Tags

details, summary, progress, meter
Short Detail

Collapsible info, progress display, measured values দেখাতে useful।

Example
<details>
  <summary>Read More</summary>
  <p>Extra content</p>
</details>
Useful Advanced

20. Advanced Media & Embed

audio, video, source, picture, iframe
Short Detail

Responsive image, external content embed, media playback-এর জন্য useful।

Example
<video controls>
  <source src="video.mp4" type="video/mp4">
</video>
Useful Advanced

21. Special Content Tags

code, pre, blockquote, q, mark, abbr, time
Short Detail

Documentation, quote, highlighted content, abbreviation, time info semanticভাবে দেখাতে।

Example
<p>Use <code>console.log()</code></p>
<abbr title="HyperText Markup Language">HTML</abbr>
Rare but Good

22. Rare but Good to Know

canvas, svg, template, noscript, address, cite
Short Detail

Graphics, reusable template, JS fallback, contact info, citation — এসব advanced scenario-তে useful।

Example
<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" fill="red"></circle>
</svg>

Interview Preparation

Revision Ready

Short concept recall interview-এর জন্য অনেক useful।

Basic Interview

27. What is HTML?

Definition

HTML = HyperText Markup Language. এটি webpage structure তৈরি করার language.

Important Interview

28. Why Semantic HTML?

Meaning + SEO + accessibility

Semantic tags content-এর meaning দেয়, SEO improve করে, accessibility ভালো করে, code maintainable করে।

Basic Interview

29. id vs class

Unique vs reusable

id unique, class reusable এবং multiple element-এ use করা যায়।

Important Interview

30. Why alt attribute is important?

Accessibility + SEO

Image describe করার জন্য, screen reader user-এর জন্য, আর search engine context বোঝার জন্য alt important।

  <meta name="description" content="HTML master practice example page">

  <header>

      <a href="#">Home</a>

        <li>HTML</li>

        <label for="email">Email</label>
        <input type="email" id="email" name="email" placeholder="Enter your email" required>

        <label for="message">Message</label>

      </form>
