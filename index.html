Essential / Must Know Tags

সবচেয়ে বেশি ব্যবহৃত

এগুলো HTML শেখার জন্য সবচেয়ে গুরুত্বপূর্ণ ট্যাগ। Basic page structure, text, link, image, form, list, table, semantic layout — সবকিছুর foundation এই group-এর মধ্যে পড়ে।

<!DOCTYPE html>
Document Type Declaration

কাজ

ব্রাউজারকে জানায় যে এটি HTML5 document।

কখন/কোথায় ব্যবহার হয়

প্রতিটি HTML file-এর একদম শুরুতে ব্যবহার হয়।

Example

<!DOCTYPE html>
<html>
Root Element

কাজ

পুরো HTML document-কে wrap করে।

কখন/কোথায় ব্যবহার হয়

DOCTYPE এর পরে, পুরো page structure এর parent element হিসেবে।

Example

<html lang="bn">
</html>
<head>
Metadata Container

কাজ

title, meta, CSS link, SEO info, script reference রাখে।

কখন/কোথায় ব্যবহার হয়

visible content-এর আগে, document setup অংশে।

Example

<head>
  <title>My Page</title>
</head>
<body>
Visible Page Content

কাজ

যে content user browser-এ দেখে, সব এখানে থাকে।

কখন/কোথায় ব্যবহার হয়

head এর পরে, visible layout তৈরির জন্য।

Example

<body>
  <h1>Hello World</h1>
</body>
<h1> - <h6>
Heading Tags

কাজ

Heading বা শিরোনাম তৈরি করে।

কখন/কোথায় ব্যবহার হয়

Page title, section heading, subsection title-এর জন্য। h1 সবচেয়ে গুরুত্বপূর্ণ।

Example

<h1>Main Title</h1>
<h2>Sub Title</h2>
<p>
Paragraph

কাজ

Paragraph text লিখতে ব্যবহার হয়।

কখন/কোথায় ব্যবহার হয়

Normal content, article text, description, note ইত্যাদিতে।

Example

<p>This is a paragraph.</p>
<a>
Anchor / Link

কাজ

Hyperlink তৈরি করে।

কখন/কোথায় ব্যবহার হয়

অন্য page, website, email, file download, page section-এ যাওয়ার জন্য।

Example

<a href="https://google.com">Visit Google</a>
<img>
Image

কাজ

Webpage-এ image দেখায়।

কখন/কোথায় ব্যবহার হয়

Photo, banner, logo, product image, illustration দেখাতে।

Example

<img src="photo.jpg" alt="My Photo">
<div>
Block Container

কাজ

একটি block-level container হিসেবে কাজ করে।

কখন/কোথায় ব্যবহার হয়

Layout, grouping, design section, CSS styling করার জন্য।

Example

<div>
  <h2>About</h2>
  <p>Some text</p>
</div>
<span>
Inline Container

কাজ

Text-এর ছোট অংশ আলাদা style বা target করার জন্য।

কখন/কোথায় ব্যবহার হয়

Inline text styling, color change, highlighting এ।

Example

<p>My color is <span style="color:red;">red</span>.</p>
<ul> / <ol> / <li>
List Tags

কাজ

Bullet list বা numbered list তৈরি করতে।

কখন/কোথায় ব্যবহার হয়

Menu, steps, features, item listing, instructions এ।

Example

<ul>
  <li>HTML</li>
  <li>CSS</li>
</ul>
<table>
Table Structure

কাজ

Row-column format-এ data দেখায়।

কখন/কোথায় ব্যবহার হয়

Result sheet, pricing chart, reports, structured data এ।

Example

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Rahim</td>
    <td>22</td>
  </tr>
</table>
<form>
Form Elements

কাজ

User input collect করার জন্য form বানানো হয়।

কখন/কোথায় ব্যবহার হয়

Login, signup, contact form, feedback, registration form এ।

Example

<form>
  <label for="name">Name</label>
  <input id="name" type="text">
  <button type="submit">Send</button>
</form>
Semantic Tags
Layout Semantics

কাজ

Page structure-কে meaningful করে তোলে। যেমন: header, nav, main, section, article, aside, footer।

কখন/কোথায় ব্যবহার হয়

Modern website layout, SEO, accessibility, clean code structure তৈরির জন্য।

Example

<header>...</header>
<nav>...</nav>
<main>
  <section>...</section>
</main>
<footer>...</footer>

Less Used but Better to Know Tags

কম ব্যবহৃত, কিন্তু জানা ভালো

এই ট্যাগগুলো সব project-এ লাগে না, কিন্তু real-world development এ অনেক useful। বিশেষ করে semantic content, media, advanced forms, quote, date/time, responsive image, embedded content ইত্যাদির জন্য।

<figure> / <figcaption>
Media with Caption

কাজ

Image, chart, diagram-এর সাথে caption দেখাতে।

কখন/কোথায় ব্যবহার হয়

Blog, article, gallery, educational content-এ।

Example

<figure>
  <img src="pic.jpg" alt="Nature">
  <figcaption>Beautiful Nature</figcaption>
</figure>
<mark> / <small> / <sup> / <sub>
Special Text Formatting

কাজ

Highlight, small note, superscript, subscript দেখাতে।

কখন/কোথায় ব্যবহার হয়

Important word highlight, copyright, math formula, chemical formula এ।

Example

<p>This is <mark>important</mark></p>
<p>x<sup>2</sup></p>
<p>H<sub>2</sub>O</p>
<code> / <pre>
Code Display

কাজ

Programming code inline বা block আকারে দেখাতে।

কখন/কোথায় ব্যবহার হয়

Tutorial, documentation, blog, note, learning material এ।

Example

<p>Use <code>console.log()</code></p>

<pre>
let x = 10;
console.log(x);
</pre>
<blockquote> / <q>
Quotation Tags

কাজ

ছোট বা বড় quote দেখাতে।

কখন/কোথায় ব্যবহার হয়

Reference, speech, inspiration quote, article citation এ।

Example

<blockquote>Learning never stops.</blockquote>
<p>He said, <q>Keep practicing</q></p>
<abbr> / <time>
Abbreviation and Time

কাজ

Short form-এর full meaning এবং date/time semantic ভাবে দেখাতে।

কখন/কোথায় ব্যবহার হয়

Blog publish date, event date, abbreviation explanation এ।

Example

<abbr title="HyperText Markup Language">HTML</abbr>
<time datetime="2025-03-20">20 March 2025</time>
<progress> / <meter>
Progress and Measurement

কাজ

Progress bar বা known range value দেখাতে।

কখন/কোথায় ব্যবহার হয়

Upload progress, task completion, disk usage, score display এ।

Example

<progress value="70" max="100"></progress>
<meter value="0.8">80%</meter>
<details> / <summary>
Expandable Content

কাজ

Click করলে content hide/show করা যায়।

কখন/কোথায় ব্যবহার হয়

FAQ, read more section, hidden explanation, collapsible notes এ।

Example

<details>
  <summary>What is HTML?</summary>
  <p>HTML is a markup language.</p>
</details>
<audio> / <video> / <source>
Media Embed

কাজ

Audio এবং video browser-এ play করতে।

কখন/কোথায় ব্যবহার হয়

Song, lecture, tutorial video, promo clip, podcast এ।

Example

<video controls width="300">
  <source src="movie.mp4" type="video/mp4">
</video>
<picture>
Responsive Image

কাজ

ভিন্ন screen size-এ ভিন্ন image serve করতে পারে।

কখন/কোথায় ব্যবহার হয়

Responsive design, performance optimization, mobile/desktop image switch এ।

Example

<picture>
  <source media="(max-width: 600px)" srcset="small.jpg">
  <img src="large.jpg" alt="Responsive Image">
</picture>
<iframe>
Embedded External Content

কাজ

অন্য webpage, map, YouTube video embed করতে।

কখন/কোথায় ব্যবহার হয়

External content দেখাতে হলে।

Example

<iframe src="https://example.com" width="300" height="200"></iframe>
<canvas> / <svg>
Graphics

কাজ

Custom graphics, vector shapes, charts, icons, animation তৈরি করতে।

কখন/কোথায় ব্যবহার হয়

Drawing, game, dashboard chart, scalable logo বা icon এ।

Example

<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" fill="red"></circle>
</svg>
<datalist> / <fieldset> / <legend>
Advanced Form Helpers

কাজ

Suggestion list, form grouping, form section title দেখাতে।

কখন/কোথায় ব্যবহার হয়

Advanced forms, grouped information, structured form design এ।

Example

<input list="browsers">
<datalist id="browsers">
  <option value="Chrome">
</datalist>
<noscript> / <template> / <address> / <cite>
Useful Advanced Tags

কাজ

JS disabled fallback, reusable template, contact info, book/movie/article title দেখাতে।

কখন/কোথায় ব্যবহার হয়

Advanced web app, contact section, citation, JS-driven content rendering এ।

Example

<noscript>Enable JavaScript</noscript>
<address>Email: test@mail.com</address>
<p>Book: <cite>Atomic Habits</cite></p>

Full Basic HTML Example

Practice Ready

নিচের example-এ HTML এর basic structure, semantic tags, text, image, list, form — সবকিছু একসাথে দেখানো হয়েছে।

Full Example
Starter HTML Page
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My First HTML Page</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>

  <header>
    <h1>My Website</h1>
    <nav>
      <a href="#">Home</a>
      <a href="#">About</a>
      <a href="#">Contact</a>
    </nav>
  </header>

  <main>
    <section>
      <h2>About Me</h2>
      <p>Hello, I am learning <strong>HTML</strong>.</p>
      <img src="myphoto.jpg" alt="My photo" width="200">
    </section>

    <section>
      <h2>Skills</h2>
      <ul>
        <li>HTML</li>
        <li>CSS</li>
        <li>JavaScript</li>
      </ul>
    </section>

    <section>
      <h2>Contact</h2>
      <form>
        <label for="name">Name:</label>
        <input type="text" id="name" placeholder="Enter your name">

        <label for="message">Message:</label>
        <textarea id="message"></textarea>

        <button type="submit">Send</button>
      </form>
    </section>
  </main>

  <footer>
    <p>© 2025 My Website</p>
  </footer>

</body>
</html>
