1. HTML Fundamentals

HTML কী, tag কী, element, attribute, nesting, comments, empty elements, block vs inline—এসব মৌলিক ধারণা না বুঝলে বাকি সবকিছু কঠিন লাগবে।

Most Use

HTML কী?

HTML-এর পূর্ণরূপ HyperText Markup Language। এটি ওয়েবপেজের structure তৈরি করার ভাষা। কোনো heading কোথায় থাকবে, paragraph কোথায় থাকবে, link, image, table, form, button, section—এসব HTML দিয়ে define করা হয়। HTML একা styling বা complex logic দেয় না; styling-এর জন্য CSS এবং behavior/interactivity-এর জন্য JavaScript ব্যবহৃত হয়। তবে HTML হলো পুরো web development-এর foundation।

উদাহরণ: 
<h1>আমার ওয়েবসাইট</h1>
<p>এটি HTML দিয়ে তৈরি structure।</p>
Most Use

Tag, Element, Content

HTML-এ tag হলো markup syntax যেমন <p> বা <div>। Element সাধারণত opening tag, content এবং closing tag-এর পুরো অংশকে বোঝায়। Content element-এর ভিতরের লেখা বা nested child হতে পারে। উদাহরণ: <p>Hello</p> এখানে p হলো tag, পুরো অংশটি paragraph element, আর Hello হলো content।

উদাহরণ: 
<p>আমি একটি paragraph element</p>
Most Use

Attribute

Attribute element-কে অতিরিক্ত তথ্য দেয়। যেমন link-এর গন্তব্য href, image path src, form field নাম name, CSS target করার জন্য class, unique identify করার জন্য id। Attribute সবসময় opening tag-এর ভিতরে লেখা হয়। Full stack development-এ HTML attribute খুবই গুরুত্বপূর্ণ কারণ backend data submission, JavaScript targeting, accessibility—সবকিছুতে এর বড় ভূমিকা আছে।

উদাহরণ: 
<a href="https://example.com" target="_blank">Visit</a>
Most Use

Nesting

একটি element-এর ভিতরে আরেকটি element রাখাকে nesting বলে। তবে nesting সবসময় সঠিকভাবে করতে হয়। Parent element-এর ভিতরে child element থাকবে এবং closing order ঠিক রাখতে হবে। ভুল nesting browser-এ unexpected behavior তৈরি করতে পারে এবং HTML validation error দেয়।

উদাহরণ: 
<div>
  <p>এখানে <strong>গুরুত্বপূর্ণ</strong> লেখা আছে।</p>
</div>
Mid Use

Comment

Comment ব্যবহার করা হয় developer note রাখার জন্য। এটি browser-এ দেখা যায় না। Project বড় হলে comment অনেক সাহায্য করে। তবে খুব obvious জিনিসে unnecessary comment না দিয়ে meaningful note লিখাই ভালো।

উদাহরণ: 
<!-- Header section starts here -->
Mid Use

Empty / Void Elements

কিছু HTML element-এর closing tag থাকে না। যেমন <br>, <hr>, <img>, <meta>, <link>, <input>। এগুলোকে void element বলা হয়। এগুলো content wrap করে না, বরং নিজে নিজেই কাজ সম্পন্ন করে।

উদাহরণ: 
<img src="photo.jpg" alt="ছবি">
<br>
<hr>
Most Use

Block vs Inline Elements

Block element সাধারণত নতুন line থেকে শুরু হয় এবং পুরো width নেয়, যেমন div, p, section, h1। Inline element একই line-এর মধ্যে থাকে, যেমন span, a, strong, em। CSS শেখার আগে এই পার্থক্য জানা খুব গুরুত্বপূর্ণ, কারণ layout behavior বোঝার ভিত্তি এটি।

উদাহরণ: 
<div>আমি block element</div>
<span>আমি inline element</span>
Less Use

Case Sensitivity & Best Syntax

HTML tag name technically case-insensitive হলেও modern practice হলো lowercase ব্যবহার করা। কারণ এতে code consistent থাকে এবং XHTML/XML বা tooling-এর সাথে কাজ করাও সহজ হয়। Attribute value quotation-এ লেখা, proper nesting, meaningful naming—এসব clean code-এর অংশ।

উদাহরণ: 
<section class="hero-banner">...</section>

2. Document Structure

একটি HTML file-এর standard structure কীভাবে তৈরি হয়, তা এখানে দেখানো হয়েছে।

Most Use

<!DOCTYPE html>

Document type declaration browser-কে জানায় যে page HTML5 standard অনুসরণ করছে। এটি না থাকলে browser quirks mode-এ যেতে পারে, ফলে rendering সমস্যা হতে পারে। সব modern HTML file-এর শুরুতেই এটি থাকা উচিত।

উদাহরণ: 
<!DOCTYPE html>
Most Use

<html>

পুরো document-এর root element হলো html। সাধারণত lang attribute দেওয়া হয় যাতে browser ও assistive technology page-এর ভাষা বুঝতে পারে। SEO ও accessibility-এর জন্য এটি খুবই উপকারী।

উদাহরণ: 
<html lang="bn">...</html>
Most Use

<head>

head অংশে metadata থাকে যা browser, search engine, social preview, CSS/JS loading, charset, responsive behavior ইত্যাদির জন্য প্রয়োজন। ব্যবহারকারী সাধারণত head-এর content সরাসরি page-এ দেখে না।

উদাহরণ: 
<head>
  <title>আমার সাইট</title>
</head>
Most Use

<body>

body-এর মধ্যে page-এর visible content থাকে। যেমন heading, paragraph, images, buttons, forms, tables, sections, cards ইত্যাদি। ব্রাউজারে user যা দেখে, তার বেশিরভাগই body-এর content।

উদাহরণ: 
<body>
  <h1>স্বাগতম</h1>
</body>

3. Metadata & Head Elements

SEO, encoding, responsive design, external resource linking—এসবের মূল জায়গা হলো head section।

Most Use

<title>

Browser tab-এ title দেখা যায় এবং search engine result-এও এর ভূমিকা রয়েছে। প্রতিটি page-এর জন্য unique, meaningful title থাকা উচিত। খুব ছোট বা খুব generic title না হওয়াই ভালো।

উদাহরণ: 
<title>Complete HTML Dashboard</title>
Most Use

<meta charset>

Character encoding define করতে charset ব্যবহার হয়। সাধারণত UTF-8 ব্যবহৃত হয় কারণ এটি বাংলা, ইংরেজি, special symbols—বেশিরভাগ character support করে। বাংলা content দেখাতে UTF-8 খুব গুরুত্বপূর্ণ।

উদাহরণ: 
<meta charset="UTF-8">
Most Use

<meta name="viewport">

Responsive design-এর জন্য viewport meta tag অপরিহার্য। এটি mobile browser-কে বলে page width device-width অনুযায়ী scale করতে। এটি না থাকলে responsive layout ভেঙে যেতে পারে।

উদাহরণ: 
<meta name="viewport" content="width=device-width, initial-scale=1.0">
Mid Use

<meta name="description">

Search engine snippet-এর জন্য page description দিতে meta description ব্যবহার করা হয়। এটি সরাসরি ranking guarantee করে না, কিন্তু click-through rate উন্নত করতে পারে। Blog, landing page, docs page—সবখানে useful।

উদাহরণ: 
<meta name="description" content="বাংলায় HTML শেখার পূর্ণ ড্যাশবোর্ড">
Most Use

<link>

External CSS file, favicon, font, preconnect এবং related metadata link করতে link tag ব্যবহার হয়। CSS file যুক্ত করার জন্য এটি খুব common। এটি void element।

উদাহরণ: 
<link rel="stylesheet" href="style.css">
Mid Use

<style>

Internal CSS লেখার জন্য style tag ব্যবহার হয়। ছোট project, demo page বা quick prototype-এ useful। তবে বড় project-এ external stylesheet ব্যবহার করাই maintainable।

উদাহরণ: 
<style>
  body { font-family: Arial; }
</style>
Less Use

<base>

সব relative URL-এর জন্য একটি base URL define করতে base tag ব্যবহৃত হয়। এটি খুব বেশি common নয়, তবে কিছু special setup-এ কাজে লাগে। ভুলভাবে ব্যবহার করলে link behavior confusion তৈরি করতে পারে।

উদাহরণ: 
<base href="https://example.com/">

4. Scripting

JavaScript যুক্ত করা, script loading control করা এবং JavaScript বন্ধ থাকলে fallback message দেখানো—এসব scripting section-এর অংশ।

Most Use

<script>

JavaScript file যুক্ত করা বা inline JavaScript লেখার জন্য script tag ব্যবহার করা হয়। এটি frontend interactivity-এর মূল মাধ্যম। Script body-এর শেষে রাখলে DOM load হওয়ার পর execute হয়, তবে defer attribute ব্যবহার করাও modern best practice।

উদাহরণ: 
<script src="app.js"></script>
Mid Use

async এবং defer

External script load behavior control করতে async এবং defer ব্যবহৃত হয়। async script download হয়ে ready হলেই execute হয়, ফলে execution order নিশ্চিত থাকে না। defer script HTML parsing শেষ হলে execute হয় এবং order maintain করে। বেশিরভাগ app script-এর জন্য defer বেশি উপযোগী।

উদাহরণ: 
<script src="app.js" defer></script>
<script src="analytics.js" async></script>
Mid Use

type="module"

Modern JavaScript module ব্যবহার করতে script-এ type="module" দেওয়া হয়। এতে import/export syntax ব্যবহার করা যায়। Advanced frontend architecture-এ এটি খুব গুরুত্বপূর্ণ।

উদাহরণ: 
<script type="module" src="main.js"></script>
Less Use

<noscript>

যদি user-এর browser-এ JavaScript disabled থাকে, তাহলে fallback content দেখানোর জন্য noscript ব্যবহার হয়। Highly JS-dependent app-এ এটি informative message দিতে সাহায্য করে।

উদাহরণ: 
<noscript>
  <p>এই ওয়েবসাইট ব্যবহার করতে JavaScript চালু করুন।</p>
</noscript>

5. Text Content Elements

Heading, paragraph, line break, section separator, preformatted text এবং long quotation—এসব text content element-এর অংশ।

Most Use

<h1> - <h6>

Heading tag content hierarchy বোঝায়। h1 সাধারণত page-এর প্রধান title, h2 section title, h3 subsection ইত্যাদি। শুধু বড় font পাওয়ার জন্য heading ব্যবহার করা ভুল; semantic structure বোঝাতে ব্যবহার করতে হবে।

উদাহরণ: 
<h1>HTML শেখা</h1>
<h2>Forms</h2>
<h3>Input Types</h3>
Most Use

<p>

সাধারণ paragraph text-এর জন্য p tag ব্যবহার হয়। Article, note, tutorial, documentation—প্রায় সব জায়গায় paragraph ব্যবহৃত হয়। Readable content তৈরি করার প্রধান ভিত্তি এটি।

উদাহরণ: 
<p>HTML হলো ওয়েবের structure তৈরির ভাষা।</p>
Most Use

<br>

একই text flow-এর মধ্যে line break দিতে br ব্যবহার হয়। Address, poem, short contact info-তে এটি useful। কিন্তু page layout spacing-এর জন্য br repeatedly ব্যবহার করা উচিত নয়।

উদাহরণ: 
<p>নাম: রাহাত<br>ঢাকা, বাংলাদেশ</p>
Mid Use

<hr>

Thematic break বা section separation বোঝাতে hr ব্যবহার করা হয়। Article-এর topic change, content divider, note segmentation-এ কাজে লাগে।

উদাহরণ: 
<p>প্রথম অংশ</p>
<hr>
<p>দ্বিতীয় অংশ</p>
Mid Use

<pre>

Preformatted text preserve করতে pre tag ব্যবহার হয়। এতে whitespace ও line break সাধারণত যেমন লেখা হয় তেমনই থাকে। Code display, poem formatting, aligned text, console output—এসব জায়গায় useful।

উদাহরণ: 
<pre>
লাইন ১
    লাইন ২
লাইন ৩
</pre>
Mid Use

<blockquote>

বড় quotation বা external উদ্ধৃতি দেখাতে blockquote ব্যবহার করা হয়। এটি semanticভাবে বোঝায় যে contentটি quoted material। Citation সহ academic বা article content-এ useful।

উদাহরণ: 
<blockquote>
  Practice makes progress.
</blockquote>

6. Inline Text Semantics

Inline text-এর meaning, emphasis, importance, code-style, quotation, abbreviation, bidirectional text, ruby annotation ইত্যাদি বোঝাতে এই tagগুলো ব্যবহৃত হয়।

Most Use

<strong> এবং <em>

strong text-এর গুরুত্ব বোঝায় এবং em text-এ emphasis বোঝায়। Browser সাধারণত strong-কে bold এবং em-কে italic দেখায়, তবে semantic অর্থই আসল। Meaningful emphasis বোঝাতে এগুলো ব্যবহার করা উচিত।

উদাহরণ: 
<p><strong>সতর্কতা:</strong> এটি গুরুত্বপূর্ণ।</p>
<p>HTML <em>ভালোভাবে</em> শিখুন।</p>
Most Use

<b>, <i>, <u>, <span>

b, i, u অনেক সময় visual styling-এর জন্য দেখা যায়, যদিও semantic use case-ও থাকতে পারে। span হলো generic inline container, যার মাধ্যমে নির্দিষ্ট text target করে style বা script apply করা যায়। Modern code-এ styling-এর জন্য CSS ব্যবহার করাই ভালো।

উদাহরণ: 
<p>এটি <span class="highlight">গুরুত্বপূর্ণ</span> অংশ।</p>
Most Use

<mark>, <small>, <sub>, <sup>

mark দিয়ে highlight, small দিয়ে ছোট informational text, sub দিয়ে নিচে থাকা text, sup দিয়ে উপরে থাকা text দেখানো হয়। Scientific notation, formula, footnote, note text, highlighted term—এসব ক্ষেত্রে useful।

উদাহরণ: 
<p>Water = H<sub>2</sub>O</p>
<p>x<sup>2</sup></p>
<p><mark>গুরুত্বপূর্ণ</mark> অংশ</p>
Mid Use

<del>, <ins>, <s>

del বোঝায় text removed হয়েছে, ins বোঝায় text inserted হয়েছে। s tag অনেক সময় এমন text বোঝাতে ব্যবহৃত হয় যা আর accurate নয়, যেমন old price। Editing history, pricing UI, revisions—এসব ক্ষেত্রে useful।

উদাহরণ: 
<p><del>500 টাকা</del> <ins>350 টাকা</ins></p>
Mid Use

<q>, <cite>, <abbr>

q inline quote-এর জন্য, cite কোনো বই/কাজ/শিল্পকর্মের reference-এর জন্য, abbr abbreviation-এর জন্য ব্যবহৃত হয়। Title attribute দিলে abbreviation-এর পূর্ণরূপ দেখানো যায়। Documentation ও article-এ useful।

উদাহরণ: 
<p>সে বলল, <q>আমি HTML শিখছি</q></p>
<p><abbr title="HyperText Markup Language">HTML</abbr></p>
Mid Use

<code>, <kbd>, <samp>, <var>

code code snippet দেখায়, kbd keyboard input বোঝায়, samp program output বোঝায়, var variable বোঝায়। Tutorial, programming note, technical documentation-এ এই tagগুলো semantic clarity দেয়।

উদাহরণ: 
<code>console.log("Hello")</code>
<kbd>Ctrl</kbd> + <kbd>S</kbd>
<var>x</var> = 10
Less Use

<dfn>, <time>, <data>

dfn কোনো term-এর defining instance বোঝাতে ব্যবহৃত হয়। time machine-readable date/time দেয়। data human-visible text-এর সাথে machine-readable value যুক্ত করে। Semantic-rich document-এ এগুলো useful।

উদাহরণ: 
<p><dfn>HTML</dfn> হলো markup language।</p>
<time datetime="2026-04-01">১ এপ্রিল ২০২৬</time>
<data value="12345">Product #12345</data>
Less Use

<bdi>, <bdo>

Bidirectional text handling-এর জন্য bdi ও bdo ব্যবহার হয়। Mixed direction language content যেমন Arabic + English + Bangla text-এ এটি useful হতে পারে। সাধারণ project-এ কম ব্যবহার হলেও internationalization context-এ গুরুত্বপূর্ণ।

উদাহরণ: 
<bdo dir="rtl">Hello</bdo>
Less Use

<ruby>, <rt>, <rp>, <wbr>

ruby annotation East Asian pronunciation note-এর জন্য ব্যবহৃত হয়। rt annotation text, rp fallback parenthesis support দেয়। wbr word break opportunity বোঝায়। Rare হলেও international text layout-এ এগুলোর ব্যবহার আছে।

উদাহরণ: 
<ruby>
  漢 <rt>kan</rt>
</ruby>
<p>longword<wbr>break</p>

7. Links & URL Schemes

Web-এর প্রাণ হলো hyperlink। Internal navigation, external link, email, phone, file download—সব anchor element দিয়ে করা যায়।

Most Use

<a> Anchor

Anchor tag link তৈরি করে। href attribute-এ destination দেওয়া হয়। এটি অন্য page, section id, external site, file link, email বা phone action—সবকিছু support করে। Navigation menu, CTA button, breadcrumb, article link—সবখানে এটি ব্যবহৃত হয়।

উদাহরণ: 
<a href="about.html">About</a>
<a href="https://example.com">External Site</a>
<a href="#forms">Forms এ যাও</a>
Most Use

mailto:, tel:, download

mailto: ব্যবহার করে email client খুলে email compose করা যায়। tel: ব্যবহার করে mobile device-এ call action trigger করা যায়। download attribute file download behavior নির্দেশ করতে পারে। Contact page-এ এগুলো খুব useful।

উদাহরণ: 
<a href="mailto:test@example.com">Email করুন</a>
<a href="tel:+880123456789">Call করুন</a>
<a href="notes.pdf" download>PDF Download</a>
Mid Use

target, rel

target="_blank" দিলে নতুন tab-এ link খুলে। rel="noopener noreferrer" security reason-এ external blank target link-এর সাথে দেওয়া ভালো। এটি performance ও security best practice।

উদাহরণ: 
<a href="https://example.com" target="_blank" rel="noopener noreferrer">
  New Tab
</a>

8. Lists

Menu, feature list, ordered steps, glossary—এসব জায়গায় list structure খুব প্রয়োজনীয়।

Most Use

<ul> + <li>

Unordered list item-এর নির্দিষ্ট ক্রম না থাকলে ব্যবহার হয়। Menu, bullet points, features, notes—এসবের জন্য common।

উদাহরণ: 
<ul>
  <li>HTML</li>
  <li>CSS</li>
</ul>
Most Use

<ol> + <li>

Ordered list তখন ব্যবহার হয় যখন item sequence গুরুত্বপূর্ণ। Tutorial steps, recipe, ranking, instructions—এসবের জন্য perfect।

উদাহরণ: 
<ol>
  <li>File তৈরি করুন</li>
  <li>HTML লিখুন</li>
</ol>
Less Use

<dl>, <dt>, <dd>

Definition list term এবং explanation দেখাতে ব্যবহৃত হয়। Dictionary-style notes, glossary, terminology page-এ useful।

উদাহরণ: 
<dl>
  <dt>HTML</dt>
  <dd>ওয়েব structure-এর ভাষা</dd>
</dl>

9. Images & Graphics

Image, responsive image, captions, image map, drawing surface এবং vector graphics—এসব এই section-এ।

Most Use

<img>

Image দেখানোর প্রধান tag হলো img। src দিয়ে file path, alt দিয়ে text alternative দেওয়া হয়। alt attribute accessibility-এর জন্য অত্যন্ত গুরুত্বপূর্ণ। Image load না হলে alt text সাহায্য করে।

উদাহরণ: 
<img src="photo.jpg" alt="একটি প্রাকৃতিক দৃশ্য">
Mid Use

<figure> + <figcaption>

কোনো image, diagram, code sample, chart-এর সাথে caption যুক্ত করতে figure ও figcaption ব্যবহার হয়। Article ও tutorial content-এ খুব সুন্দর semantic structure দেয়।

উদাহরণ: 
<figure>
  <img src="logo.png" alt="লোগো">
  <figcaption>কোম্পানির লোগো</figcaption>
</figure>
Mid Use

<picture> + <source>

Responsive image বা different image format support করার জন্য picture ব্যবহার করা হয়। বিভিন্ন screen size বা format অনুযায়ী browser সঠিক source বেছে নিতে পারে। Performance optimization-এর জন্য useful।

উদাহরণ: 
<picture>
  <source srcset="small.jpg" media="(max-width: 600px)">
  <img src="large.jpg" alt="Responsive image">
</picture>
Less Use

<map> + <area>

Image map তৈরি করে image-এর নির্দিষ্ট অঞ্চলকে clickable করা যায়। Modern UI-তে কম ব্যবহৃত হলেও কিছু educational বা custom diagram-based interface-এ এর দরকার হতে পারে।

উদাহরণ: 
<img src="planets.jpg" usemap="#planetmap" alt="Planets">
<map name="planetmap">
  <area shape="rect" coords="0,0,100,100" href="mars.html" alt="Mars">
</map>
Mid Use

<canvas>

Canvas একটি drawing surface দেয় যার উপর JavaScript দিয়ে graphics, animation, game, charts, effects আঁকা যায়। Raw HTML হিসেবে canvas শুধু area তৈরি করে, আসল drawing JavaScript দিয়ে হয়।

উদাহরণ: 
<canvas id="myCanvas" width="300" height="150"></canvas>
Mid Use

<svg>

SVG হলো scalable vector graphics, যা resolution-independent। Logo, icon, chart, custom illustration-এর জন্য খুব উপযোগী। CSS ও JavaScript দিয়ে manipulate করা যায়।

উদাহরণ: 
<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" fill="tomato" />
</svg>

10. Audio & Video

Multimedia content embed করার জন্য audio, video, source এবং subtitle track গুরুত্বপূর্ণ।

Mid Use

<audio>

Audio file play করতে audio tag ব্যবহৃত হয়। controls দিলে browser player দেখায়। Podcast, pronunciation training, music preview, spoken lesson—এসব ক্ষেত্রে useful।

উদাহরণ: 
<audio controls>
  <source src="song.mp3" type="audio/mpeg">
</audio>
Mid Use

<video>

Video content embed করতে video tag ব্যবহৃত হয়। controls, autoplay, muted, loop, poster ইত্যাদি attribute থাকতে পারে। Course site, demo page, portfolio, media platform-এ useful।

উদাহরণ: 
<video controls width="320">
  <source src="lesson.mp4" type="video/mp4">
</video>
Less Use

<track>

Subtitle, caption, chapter data দেওয়ার জন্য track ব্যবহৃত হয়। Accessibility এবং multilingual learning experience-এর জন্য এটি খুব helpful।

উদাহরণ: 
<video controls>
  <source src="movie.mp4" type="video/mp4">
  <track src="subtitles.vtt" kind="subtitles" srclang="en" label="English">
</video>

11. Tables

Structured data row-column আকারে দেখানোর জন্য table element family ব্যবহৃত হয়। Layout করার জন্য table ব্যবহার করা উচিত নয়।

Most Use

<table>, <tr>, <th>, <td>

Table-এর মূল building block হলো table, tr, th, td। tr = row, th = header cell, td = data cell। Marksheet, pricing table, schedule, report data—এসবের জন্য essential।

উদাহরণ: 
<table border="1">
  <tr>
    <th>নাম</th>
    <th>স্কোর</th>
  </tr>
  <tr>
    <td>রিমা</td>
    <td>90</td>
  </tr>
</table>
Mid Use

<thead>, <tbody>, <tfoot>

Large table-এ semantic grouping করার জন্য thead, tbody, tfoot ব্যবহার করা হয়। এতে structure স্পষ্ট হয়, styling সহজ হয় এবং assistive technology table better বুঝতে পারে।

উদাহরণ: 
<table>
  <thead><tr><th>পণ্য</th><th>দাম</th></tr></thead>
  <tbody><tr><td>Mouse</td><td>500</td></tr></tbody>
</table>
Mid Use

<caption>, colspan, rowspan

Table title বোঝাতে caption ব্যবহার হয়। colspan দিয়ে multiple column merge করা যায়, rowspan দিয়ে multiple row merge করা যায়। Complex table-এ এগুলো দরকার হতে পারে।

উদাহরণ: 
<table border="1">
  <caption>ফলাফল</caption>
  <tr><th colspan="2">Student Info</th></tr>
  <tr><td>রাহাত</td><td>A+</td></tr>
</table>
Less Use

<colgroup> এবং <col>

Table column group define করতে colgroup এবং specific column property target করতে col ব্যবহার হয়। Styling ও structural hint-এর জন্য useful, তবে beginner project-এ কম দেখা যায়।

উদাহরণ: 
<table>
  <colgroup>
    <col span="1" style="background:#f3f4f6">
  </colgroup>
</table>

12. Forms & Form Controls

Full stack development-এর সবচেয়ে গুরুত্বপূর্ণ chapterগুলোর একটি। User input নেওয়া, validate করা, backend-এ পাঠানো—এসব form-এর মাধ্যমে হয়।

Most Use

<form>

Form হলো user data collect করার wrapper। action বলে কোথায় data যাবে, method বলে GET না POST হবে। enctype file upload-এর ক্ষেত্রে দরকার হতে পারে। Backend integration বুঝতে form খুবই গুরুত্বপূর্ণ।

উদাহরণ: 
<form action="/submit" method="post">
  ...
</form>
Most Use

<label>

Form field-এর description বোঝাতে label ব্যবহার হয়। for attribute input id-এর সাথে match করলে usability ও accessibility অনেক ভালো হয়। ভালো HTML form মানেই label correctly ব্যবহার করা।

উদাহরণ: 
<label for="email">ইমেইল</label>
<input id="email" type="email" name="email">
Most Use

<input> basics

input হলো সবচেয়ে versatile form element। type অনুযায়ী behavior বদলায়। name attribute backend-এর জন্য জরুরি, placeholder hint দেয়, required validation trigger করে, value default value সেট করে।

উদাহরণ: 
<input type="text" name="username" placeholder="আপনার নাম">
Most Use

Common Input Types

text, email, password, number, checkbox, radio, file, date, time, tel, url, search—এসব input type খুব common। সঠিক type ব্যবহার করলে browser built-in validation, mobile keyboard optimization এবং better UX দেয়।

উদাহরণ: 
<input type="text">
<input type="email">
<input type="password">
<input type="number">
<input type="checkbox">
<input type="radio">
Mid Use

More Input Types

color, range, datetime-local, month, week, hidden, submit, reset, button, image—এসব input type specific কাজের জন্য ব্যবহার হয়। সব project-এ না লাগলেও HTML reference হিসেবে জানা দরকার।

উদাহরণ: 
<input type="date">
<input type="range" min="0" max="100">
<input type="color">
<input type="hidden" name="token" value="abc123">
Most Use

<textarea>

Multi-line text input নেওয়ার জন্য textarea ব্যবহৃত হয়। Message box, comments, description, address, review ইত্যাদিতে খুব common।

উদাহরণ: 
<textarea name="message" rows="4" cols="30"></textarea>
Most Use

<button>

Button click action trigger করে। type="submit", type="reset", type="button"—এই তিনটি common form usage আছে। Semantic button tag div click handler-এর চেয়ে better practice।

উদাহরণ: 
<button type="submit">Submit</button>
Most Use

<select>, <option>, <optgroup>

Dropdown select করার জন্য select ব্যবহার হয়। option প্রতিটি option define করে। অনেক option থাকলে optgroup দিয়ে group করা যায়। Country, category, course, status selector-এ common।

উদাহরণ: 
<select name="course">
  <optgroup label="Frontend">
    <option value="html">HTML</option>
  </optgroup>
</select>
Mid Use

<fieldset> + <legend>

Related form controls group করার জন্য fieldset এবং group title-এর জন্য legend ব্যবহার হয়। বড় form-এ readability বাড়ে এবং accessibility উন্নত হয়।

উদাহরণ: 
<fieldset>
  <legend>ব্যক্তিগত তথ্য</legend>
  <input type="text" name="name">
</fieldset>
Less Use

<datalist>

input-এর জন্য predefined suggestion list দিতে datalist ব্যবহার হয়। এটি autocomplete-এর lightweight semantic option। Search suggestion বা browser suggestion type UX-এ useful হতে পারে।

উদাহরণ: 
<input list="browsers">
<datalist id="browsers">
  <option value="Chrome">
  <option value="Firefox">
</datalist>
Less Use

<output>, <progress>, <meter>

output calculated result দেখাতে পারে, progress task progress বোঝায়, meter known range-এর measurement বোঝায়। এগুলো specific use case-এ ভালো semantic meaning দেয়।

উদাহরণ: 
<progress value="60" max="100"></progress>
<meter value="0.7">70%</meter>
ফর্মে গুরুত্বপূর্ণ attribute: name, id, value, placeholder, required, readonly, disabled, checked, selected, minlength, maxlength, min, max, step, autocomplete, autofocus, multiple, accept, pattern।

13. Semantic Layout Elements

Modern HTML-এর সবচেয়ে গুরুত্বপূর্ণ দিকগুলোর একটি হলো semantic layout। এতে SEO, accessibility ও code readability অনেক ভালো হয়।

Most Use

<header>, <nav>, <main>, <footer>

header সাধারণত page intro বা top area, nav navigation area, main page-এর primary content, footer নিচের information section বোঝায়। Proper page structure গঠনে এগুলো অত্যন্ত গুরুত্বপূর্ণ।

উদাহরণ: 
<header>Logo</header>
<nav>Menu</nav>
<main>Main Content</main>
<footer>Footer</footer>
Most Use

<section> এবং <article>

section related content group করে, article standalone meaningful content বোঝায়। Blog post, card article, knowledge block, landing page section—এসব ক্ষেত্রে এগুলো ideal।

উদাহরণ: 
<section>
  <h2>Features</h2>
</section>

<article>
  <h2>Blog Title</h2>
</article>
Mid Use

<aside>

Main content-এর পাশে related sidebar, tips, ads, extra notes বা related links দেখাতে aside ব্যবহার হয়। Documentation page, blog layout, learning dashboard-এ useful।

উদাহরণ: 
<aside>
  <h3>Related Resources</h3>
</aside>
Less Use

<address>

address সাধারণত contact information বোঝাতে ব্যবহৃত হয়। এটি physical address না হলেও author/contact information represent করতে পারে। Footer বা contact section-এ ব্যবহার হতে পারে।

উদাহরণ: 
<address>
  ঢাকা, বাংলাদেশ
</address>
Most Use

<div>

div semantic tag নয়, তবে generic block container হিসেবে খুব common। CSS layout, wrapper, card, grid item, grouped element—এসব ক্ষেত্রে div ব্যাপকভাবে ব্যবহৃত হয়। Semantic alternative থাকলে সেটি আগে ভাবা ভালো।

উদাহরণ: 
<div class="card">
  <h3>Title</h3>
</div>

14. Embedded Content

External page, PDF, media object বা third-party widget embed করতে এই tagগুলো ব্যবহার হয়।

Mid Use

<iframe>

অন্য webpage বা service আপনার page-এর ভিতরে embed করতে iframe ব্যবহার হয়। YouTube, Google Maps, external tools, payment UI—এসব embed করতে common। Security-এর জন্য sandbox attribute সম্পর্কে ধারণা রাখা ভালো।

উদাহরণ: 
<iframe src="https://example.com" width="300" height="200"></iframe>
Less Use

iframe sandbox basics

sandbox attribute iframe content-এর capability সীমিত করতে পারে। Third-party embed-এর ক্ষেত্রে security risk কমাতে এটি useful। Advanced topic হলেও জানা দরকার।

উদাহরণ: 
<iframe src="https://example.com" sandbox="allow-scripts"></iframe>
Less Use

<embed>, <object>, <param>

Legacy বা special content embed করতে embed/object ব্যবহৃত হতে পারে। Modern web-এ কম common, কিন্তু পুরনো project-এ দেখা যায়। object-এর child হিসেবে param ব্যবহার করা হতো additional parameter দিতে।

উদাহরণ: 
<object data="file.pdf" type="application/pdf" width="400" height="300">
  <param name="zoom" value="100">
</object>

15. Interactive Elements

HTML-এর কিছু built-in interactive tag আছে যা JavaScript ছাড়াই basic UI behavior দিতে পারে।

Mid Use

<details> + <summary>

Expand/collapse content বানাতে details ও summary ব্যবহার হয়। FAQ section, hidden note, extra explanation, collapsible topic list—এসবের জন্য excellent।

উদাহরণ: 
<details>
  <summary>HTML কী?</summary>
  <p>HTML হলো markup language.</p>
</details>
Less Use

<dialog>

Dialog tag modal বা pop-up style content তৈরি করতে পারে। Modern হলেও সব project-এ native dialog ব্যবহার করা হয় না; অনেক সময় custom modal বানানো হয়। তবুও reference হিসেবে জানা দরকার।

উদাহরণ: 
<dialog open>
  <p>এটি একটি dialog box</p>
</dialog>

16. Template & Web Component Related Elements

Reusable markup, client-side rendering এবং Web Components ecosystem-এ template-related tag জানা advanced learner-এর জন্য উপকারী।

Less Use

<template>

Template tag browser-এ render হয় না, তবে JavaScript দিয়ে clone করে DOM-এ inject করা যায়। Reusable markup fragment তৈরিতে useful। Advanced frontend বা component-based logic-এ এটি সাহায্য করে।

উদাহরণ: 
<template id="cardTemplate">
  <div class="card">Hello</div>
</template>
Less Use

<slot>

Slot Web Components-এ content projection-এর জন্য ব্যবহৃত হয়। Beginner HTML learning-এ খুব জরুরি নয়, কিন্তু expert reference হিসেবে জানা উচিত।

উদাহরণ: 
<slot name="title"></slot>

17. Global Attributes

কিছু attribute প্রায় সব HTML element-এ ব্যবহার করা যায়। এগুলো styling, scripting, accessibility, interaction—সব জায়গায় গুরুত্বপূর্ণ।

Most Use

id, class, title, style

id unique identify করে, class group styling বা JS targeting-এর জন্য ব্যবহার হয়, title tooltip-type extra info দিতে পারে, style inline CSS-এর জন্য। এর মধ্যে id ও class সবচেয়ে বেশি ব্যবহৃত।

উদাহরণ: 
<div id="hero" class="section-card" title="Hero Section"></div>
Most Use

lang, dir, hidden

lang document/content language বোঝায়, dir text direction নিয়ন্ত্রণ করে, hidden element সাময়িকভাবে hide করতে পারে। Multilingual ও accessible project-এ এগুলো গুরুত্বপূর্ণ।

উদাহরণ: 
<p lang="bn">বাংলা লেখা</p>
<div hidden>Hidden Content</div>
Mid Use

tabindex, contenteditable, draggable

tabindex keyboard navigation control করে, contenteditable element-কে editable বানাতে পারে, draggable drag behavior enable করতে পারে। Special UI behavior-এ useful।

উদাহরণ: 
<div tabindex="0">Focusable</div>
<p contenteditable="true">এডিট করা যাবে</p>
Less Use

spellcheck, translate

spellcheck browser spell checking control করে। translate attribute translation behavior hint দিতে পারে। Special content ক্ষেত্রে useful হতে পারে।

উদাহরণ: 
<textarea spellcheck="true"></textarea>
<span translate="no">HTML</span>
Most Use

data-* attributes

Custom data store করার জন্য data-* attribute খুব useful। JavaScript দিয়ে element-এর সাথে extra data attach করতে এটি commonly ব্যবহৃত হয়। Frontend components, filtering, interactive UI-তে প্রায়ই দেখা যায়।

উদাহরণ: 
<button data-id="101" data-role="delete">Delete</button>

18. Accessibility Basics

Professional HTML লিখতে accessibility অপরিহার্য। এটি শুধু bonus skill নয়, বরং ভালো web development-এর অংশ।

Most Use

alt, label, heading order, semantic HTML

Accessibility-এর মৌলিক নিয়মগুলোর মধ্যে আছে image-এ alt text, form field-এ label, proper heading hierarchy, semantic section usage, keyboard-friendly interaction। এগুলো screen reader ব্যবহারকারীদের জন্য খুব গুরুত্বপূর্ণ।

উদাহরণ: 
<img src="team.jpg" alt="আমাদের টিমের গ্রুপ ছবি">
<label for="name">নাম</label>
<input id="name" type="text">
Mid Use

ARIA basics

Native HTML যথেষ্ট না হলে ARIA attribute screen reader-কে extra meaning দেয়। তবে মনে রাখতে হবে: “No ARIA is better than bad ARIA.” যতটা সম্ভব semantic HTML আগে ব্যবহার করো। ARIA advanced layer হিসেবে ভাবা ভালো।

উদাহরণ: 
<button aria-label="মেনু খুলুন">☰</button>

19. HTML Entities & Special Characters

কিছু character সরাসরি HTML-এ লিখলে conflict হতে পারে। তখন entity ব্যবহার করা হয়।

Mid Use

Common Entities

<, >, & এই character-গুলো HTML syntax-এর অংশ হওয়ায় content হিসেবে দেখাতে entity ব্যবহার করতে হয়। এছাড়া non-breaking space, copyright, trademark ইত্যাদির জন্যও entity ব্যবহৃত হয়।

উদাহরণ: 
&lt;  = <
&gt;  = >
&amp; = &
&nbsp; = non-breaking space
&copy; = ©

20. Obsolete / Deprecated Elements

পুরনো HTML-এ ব্যবহৃত কিন্তু modern HTML-এ discouraged বা deprecated tag জানা দরকার, যাতে legacy code দেখলে চিনতে পারেন।

Less Use

<font>, <center>, <marquee>

আগে text color/size, alignment, scrolling text-এর জন্য এসব tag ব্যবহৃত হতো। এখন CSS দিয়ে এসব কাজ করা হয়। Modern project-এ এগুলো ব্যবহার করা উচিত নয়।

উদাহরণ: 
<!-- Deprecated -->
<center>পুরনো পদ্ধতি</center>
Less Use

<acronym>, <big>, <tt>

acronym-এর জায়গায় abbr, big-এর জায়গায় CSS font-size, tt-এর জায়গায় code বা CSS monospace ব্যবহৃত হয়। Legacy tutorial-এ এগুলো দেখতে পারেন।

উদাহরণ: 
<!-- Deprecated -->
<big>Old Text</big>
Less Use

<frameset>, <frame>, <noframes>

পুরনো websites-এ page frame-based layout-এর জন্য frameset system ব্যবহার হতো। এখন এটি obsolete। Modern web-এ iframe ছাড়া frame-based layout ব্যবহার করা হয় না।

উদাহরণ: 
<!-- Obsolete frame system -->

21. HTML Best Practices

একজন beginner থেকে professional developer হতে হলে শুধু tag মুখস্থ করলে হবে না; clean, semantic, maintainable HTML লিখতে হবে।

Most Use

Use Semantic HTML First

সবকিছু div দিয়ে না বানিয়ে header, nav, main, section, article, footer ইত্যাদি semantic tag ব্যবহার করো। এতে project maintain করা সহজ হয় এবং SEO/accessibility উন্নত হয়।

উদাহরণ: 
<main>
  <article>
    <h2>Post Title</h2>
  </article>
</main>
Most Use

Always Write Clean Structure

Proper indentation, meaningful class naming, logical nesting, smaller sections—এসব maintainability বাড়ায়। Clean HTML team project-এ অনেক গুরুত্বপূর্ণ।

উদাহরণ: 
<section class="hero">
  <h1>Welcome</h1>
  <p>Intro text</p>
</section>
Most Use

Accessibility Matters

Image-এ alt, form-এ label, button-এ meaningful text, keyboard usability, correct heading order—এসব ভালো practice নয় শুধু, বরং responsible web development-এর অংশ।

উদাহরণ: 
<button type="submit">রেজিস্টার করুন</button>
Mid Use

Separate Structure, Style, Behavior

HTML structure-এর জন্য, CSS style-এর জন্য, JavaScript behavior-এর জন্য। এই separation বজায় রাখলে code clean থাকে, debug করা সহজ হয়, reuse করা যায়।

উদাহরণ: 
<p class="warning">সতর্কবার্তা</p>
Mid Use

Validate Your HTML

W3C Validator দিয়ে HTML check করলে unclosed tag, invalid nesting, wrong attribute usage ইত্যাদি ধরা যায়। Beginner-দের জন্য validation খুব helpful habit।

উদাহরণ: 
https://validator.w3.org/
