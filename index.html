Document Structure

Must Master
Highest Priority Most Used

1. Basic HTML Document Structure

doctype, html, head, body, title, meta, link, script
Detailed Essential Note

একটি HTML page-এর মূল skeleton এই structure দিয়ে তৈরি হয়। Browser document কীভাবে interpret করবে, page title কী হবে, metadata কোথায় থাকবে, visible content কোথায় থাকবে — সবকিছু এই foundation-এর উপর দাঁড়ায়।

When / Where to Use

প্রতিটি HTML file-এর শুরুতে এই structure ব্যবহার করা হয়। এটা ছাড়া clean page setup হয় না।

Pro Tip / Common Mistake

অনেক beginner body/head order গুলিয়ে ফেলে বা meta charset দিতে ভুলে যায়। শুরু থেকেই complete structure habit করো।

Example
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My Page</title>
</head>
<body>
  <h1>Hello</h1>
</body>
</html>
Highest Priority Most Used

2. Head vs Body

metadata vs visible content
Detailed Essential Note

head অংশে metadata থাকে — title, meta, CSS links, scripts reference। body অংশে user যেটা browser-এ দেখে, সেই visible content থাকে।

When / Where to Use

Page setup-related info head-এ, actual content body-তে রাখো।

Pro Tip / Common Mistake

Visible text বা image কখনো head-এর ভিতরে দিও না। Head setup-এর জন্য, body content-এর জন্য।

Example
<head>
  <title>Portfolio</title>
</head>

<body>
  <h1>Welcome</h1>
</body>

Text & Content

Must Master
Highest Priority Most Used

3. Headings, Paragraph, Strong, Em

h1-h6, p, strong, em
Detailed Essential Note

Headings document hierarchy তৈরি করে, paragraph readable text group করে, strong important meaning দেয়, em emphasis দেয়। শুধু style না, semantic meaning-ও এখানে important।

When / Where to Use

Blog, article, landing page, card content — সব জায়গায় heading ও paragraph লাগে।

Pro Tip / Common Mistake

Random heading level jump কোরো না। h1 → h2 → h3 logical order maintain করো।

Example
<h1>Main Title</h1>
<p>This is <strong>important</strong> text.</p>
<p>I <em>really</em> like HTML.</p>
Highest Priority Most Used

4. Div, Span, Br, Hr

containers and line/separator tags
Detailed Essential Note

div block-level container, span inline container। br line break দেয়, hr section separator দেয়। এগুলো layout এবং text grouping-এ খুব common।

When / Where to Use

Div দিয়ে section group করো, span দিয়ে text-এর ছোট অংশ target করো।

Pro Tip / Common Mistake

সবকিছু div দিয়ে ভরিয়ে ফেলো না। যেখানে semantic tag দরকার, সেখানে semantic tag use করো।

Example
<div>
  <p>My favorite color is <span>blue</span>.</p>
  <hr>
</div>

Links & Media Basics

Must Master
Highest Priority Most Used

5. Anchor Tag

a, href
Detailed Essential Note

Anchor tag hyperlink তৈরি করে। Website navigation, external website, section jump, email link, downloadable file — সবকিছুর জন্য anchor essential।

When / Where to Use

Menu links, footer links, buttons-as-links, internal page navigation — everywhere।

Pro Tip / Common Mistake

Link target দেওয়ার জন্য href use হয়, src না। এটা beginner-দের common mistake।

Example
<a href="https://example.com">Visit Site</a>
Highest Priority Most Used

6. Image Tag

img, src, alt
Detailed Essential Note

Image add করার জন্য img use হয়। src file path দেয়, alt image describe করে। Alt accessibility এবং SEO — দুইটার জন্যই very important।

When / Where to Use

Logo, profile photo, product image, banner, illustration—সব media image use case-এ।

Pro Tip / Common Mistake

img tag-এ href ব্যবহার করো না। আর meaningful alt text দিতে ভুলবে না।

Example
<img src="photo.jpg" alt="Student learning HTML">

7. Ordered & Unordered Lists

ul, ol, li
Detailed Essential Note

List structured item group দেখানোর সবচেয়ে simple semantic way। Unordered list bullet-based, ordered list number-based।

When / Where to Use

Menu, features, steps, task list, requirements list — সবখানে।

Pro Tip / Common Mistake

List item সবসময় li-এর ভিতরে রাখো। ul বা ol-এর direct plain text লিখো না।

Example
<ol>
  <li>Learn HTML</li>
  <li>Practice Daily</li>
</ol>
Important Mid Use

8. Definition List

dl, dt, dd
Detailed Essential Note

Definition list term এবং তার explanation semanticভাবে দেখাতে use হয়। এটা সাধারণ list-এর মতো না — glossary type content-এর জন্য better।

When / Where to Use

Glossary, FAQ-like term definition, metadata display-তে useful।

Pro Tip / Common Mistake

Normal bullet list-এর জায়গায় definition list use করো না। Use case অনুযায়ী tag choose করো।

Example
<dl>
  <dt>HTML</dt>
  <dd>Markup language for web pages</dd>
</dl>

9. Basic Table Structure

table, tr, th, td, caption, thead, tbody
Detailed Essential Note

Table row-column data show করার জন্য use হয়। Caption table title দেয়, th header cell, td data cell। Structured tabular data-এর জন্য table best।

When / Where to Use

Result sheet, pricing table, report, schedule, score table — এসব জায়গায়।

Pro Tip / Common Mistake

Layout করার জন্য table use করো না। Table শুধু real tabular data-এর জন্য।

Example
<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Rahim</td>
    <td>20</td>
  </tr>
</table>
Important Mid Use

10. Table Sections

thead, tbody, tfoot
Detailed Essential Note

Table-এর logical parts আলাদা করতে thead, tbody, tfoot useful। এতে code readable হয় এবং bigger table manage করা সহজ হয়।

When / Where to Use

Large table, report table, summary row, footer row — এসব ক্ষেত্রে।

Pro Tip / Common Mistake

Small table-এ না লাগলেও বড় data table-এ structure maintain করা ভালো।

Example
<table>
  <thead>
    <tr><th>Item</th></tr>
  </thead>
  <tbody>
    <tr><td>Book</td></tr>
  </tbody>
</table>

Forms & Inputs

Must Master
Highest Priority Most Used

11. Core Form Elements

form, input, label, button, textarea, select, option
Detailed Essential Note

User input collect করার জন্য form system use হয়। input single-line data নেয়, textarea multi-line data নেয়, select dropdown choice দেয়, label field explain করে।

When / Where to Use

Login, signup, contact form, feedback form, search form — সবখানে।

Pro Tip / Common Mistake

Input-এর সাথে label link করতে for/id use করো। এটা accessibility-এর জন্য important।

Example
<form>
  <label for="email">Email</label>
  <input id="email" type="email">
  <button type="submit">Send</button>
</form>
Highest Priority Most Used

12. Important Input Types

text, email, password, checkbox, radio, file, date
Detailed Essential Note

Input type field-এর purpose define করে। Email field email validation hint দেয়, password hidden input দেয়, checkbox multiple choice, radio single choice, file upload file নেয়।

When / Where to Use

Registration, settings panel, survey, upload form — সব জায়গায় use হয়।

Pro Tip / Common Mistake

সব input-এ type="text" use না করে correct type use করো। এতে usability improve হয়।

Example
<input type="text">
<input type="email">
<input type="checkbox">
<input type="date">
Very Important Mid Use

13. Form Submission & Grouping

action, method, enctype, fieldset, legend, datalist
Detailed Essential Note

Form কোথায় যাবে, কীভাবে data পাঠাবে, আর form sections কিভাবে group হবে — তা action, method, fieldset, legend-এর মাধ্যমে control হয়।

When / Where to Use

Real backend-connected forms বা structured forms-এ।

Pro Tip / Common Mistake

File upload থাকলে enctype ভুলে যেও না। আর related inputs group করতে fieldset useful।

Example
<form action="/submit" method="post">
  <fieldset>
    <legend>Personal Info</legend>
    <input type="text">
  </fieldset>
</form>

Attributes

Must Master
Highest Priority Most Used

14. Core Attributes

href, src, alt, id, class, title
Detailed Essential Note

Attributes tag-কে extra meaning বা functionality দেয়। href link target দেয়, src file source দেয়, alt image description দেয়, class/id CSS ও JS targeting-এ useful।

When / Where to Use

Link, image, styling, scripting, tooltip text — সব basic web work-এ।

Pro Tip / Common Mistake

href vs src confuse করো না। আর alt attribute empty না রেখে meaningful রাখো।

Example
<a href="about.html" class="nav-link">About</a>
<img src="logo.png" alt="Website logo">
Highest Priority Most Used

15. Form Attributes

name, value, placeholder, required, checked, selected, type
Detailed Essential Note

Form field-এর behavior control করতে এই attributes দরকার। Required mandatory field করে, placeholder hint দেয়, checked/selected default selected state দেয়।

When / Where to Use

সব practical form-এ এই attributes useful।

Pro Tip / Common Mistake

Placeholder label-এর replacement না। Label use করতেই হবে।

Example
<input type="text" name="username" placeholder="Enter name" required>
<input type="checkbox" checked>
Important Mid Use

16. Global Attributes

lang, dir, hidden, tabindex, data-*
Detailed Essential Note

অনেক HTML element-এই use করা যায় এমন common attributes-কে global attributes বলে। এগুলো accessibility, behavior, data binding-এ useful।

When / Where to Use

Language declaration, hidden content, JS custom data storage, keyboard order control-এ।

Pro Tip / Common Mistake

Data attributes custom info store করতে useful, but unnecessary random use avoid করো।

Example
<p lang="bn" data-topic="html">বাংলা লেখা</p>

17. Layout Semantics

header, nav, main, footer
Detailed Essential Note

Semantic tags content-এর meaning clearly বোঝায়। Header top section, nav navigation links, main main content, footer bottom info represent করে।

When / Where to Use

প্রায় সব modern website layout-এ।

Pro Tip / Common Mistake

সব layout div দিয়ে না বানিয়ে meaningful semantic tag use করো।

Example

<nav>Menu</nav>
<main>Main Content</main>
<footer>Footer</footer>
Very Important Mid Use

18. Content Semantics

section, article, aside, figure, figcaption
Detailed Essential Note

Section related content group করে, article independent content, aside side content, figure media group, figcaption media caption দেয়।

When / Where to Use

Blog, news page, content-heavy page, documentation, learning site-এ।

Pro Tip / Common Mistake

Every div-কে section বানিয়ে ফেলো না। Real logical grouping থাকলে semantic tag use করো।

Example
<section>
  <article>Blog post</article>
  <aside>Related links</aside>
</section>
